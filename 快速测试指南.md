## 🔍 如何查看测试结果

### 1. 浏览器控制台
按 `F12` → Console 标签

**成功的日志：**
```
✅ 注册成功: {...}
✅ 订单创建成功: {...}
✅ 充值成功: {...}
```

**失败的日志：**
```
❌ 注册失败: 错误信息
❌ 订单创建失败: 错误信息
```

### 2. Network 标签
按 `F12` → Network 标签

**查看请求：**
- 请求 URL
- 请求方法（POST/GET）
- 状态码（200/201/400/500）
- Request Payload（发送的数据）
- Response（返回的数据）

### 3. Supabase 数据库
打开：https://supabase.com/dashboard

**查看数据：**
```sql
-- 查看用户
SELECT * FROM users ORDER BY created_at DESC LIMIT 10;

-- 查看订单
SELECT * FROM orders ORDER BY created_at DESC LIMIT 10;

-- 查看评价
SELECT * FROM reviews ORDER BY created_at DESC LIMIT 10;

-- 查看钱包交易
SELECT * FROM wallet_transactions ORDER BY created_at DESC LIMIT 10;
```

## 🐛 常见问题

### 问题 1: 请求失败 404
**原因：** API 路由不存在

**解决：**
1. 检查后端是否部署
2. 访问：https://luxe-pi-kohl.vercel.app/health
3. 确认返回 `{ status: "ok" }`

### 问题 2: 请求失败 401
**原因：** Token 无效或过期

**解决：**
1. 重新登录
2. 检查 localStorage 中的 token
3. 确认 token 格式正确

### 问题 3: 请求失败 500
**原因：** 后端服务器错误

**解决：**
1. 查看后端日志
2. 检查数据库连接
3. 验证请求数据格式

### 问题 4: CORS 错误
**原因：** 跨域配置问题

**解决：**
后端已配置 CORS，包含 `http://localhost:5173`
